C++ 中结构和类有什么区别？
**(A)** 一个结构的所有成员都是公共的，结构没有构造函数和析构函数
**(B)** 一个类的成员默认是私有的，struct 的成员默认是公共的。从类/结构派生结构时，基类/结构的默认访问说明符是公共的，派生类时，默认访问说明符是私有的。
**(C)** 一个结构的所有成员都是公共的，结构没有虚函数
**(D)** 以上所有

**答案:****(B)**
