以下程序的输出？

```
#include<stdio.h>
int main()
{
    float x = 0.1;
    if ( x == 0.1 )
        printf("IF");
    else if (x == 0.1f)
        printf("ELSE IF");
    else
        printf("ELSE");
}
```

**(A)**ELSE IF
**(B)**IF
**(C)**ELSE

**回答:****(A)**
