## 卡片打印处理机（Card Print Processor 的缩写）

```cpp
#include <iostream>
#include <cstring>
using namespace std;

int main()
{
    char s1[] = "geeksforgeeksforgeeks";
    char s2 = 'f';
    char *ptr = strchr( s1, s2);
    cout << ptr;
    return 0;
}
```

**输出:**

```cpp
forgeeksforgeeks
```

*   **描述:** strchr( str，c)返回一个指向字符串中第一个出现的字符‘c’的指针。这里 s2 是“f”，strchr()返回它在 s1 中第一次出现的地址。
