会调用析构函数吗，如果会，那么是由于哪个向量？

```
#include <iostream>
#include <vector>
using namespace std;

class a
{
public :
    ~a()
    {
        cout << "destroy";
    }
};
int main()
{
   vector <a*> *v1  = new vector<a*>;
   vector <a> *v2  = new vector<a>;
   return 0;
}
```

**(A)**v1
**(B)**v2
**(C)**v1 和 v2
**(D)** 无析构函数调用

**答案:****(D)**
