```cpp
#include<iostream>
using namespace std;

class Base
{
public:
    virtual void show() = 0;
};

class Derived: public Base
{
public:
    void show() { cout<<"In Derived \n"; }
};

int main(void)
{
    Derived d;
    Base &br = d;
    br.show();
    return 0;
}
```

**(A)** 编译器在“Base & br = d”行出错
**(B)** 空输出
**(C)** 在衍生

**回答:****(C)**
