**例 1:重载++运算符**

## 卡片打印处理机（Card Print Processor 的缩写）

```cpp
// CPP program to illustrate
// operators that can be overloaded
#include <iostream>
using namespace std;

class overload {
private:
    int count;

public:
    overload()
        : count(4)
    {
    }

    void operator++() { count = count + 1; }
    void Display() { cout << "Count: " << count; }
};

int main()
{
    overload i;
    // this calls "function void operator ++()" function
    ++i;
    i.Display();
    return 0;
}
```

**Output**

```cpp
Count: 5
```
