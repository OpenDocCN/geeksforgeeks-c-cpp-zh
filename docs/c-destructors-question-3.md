# C++ |析构函数|问题 3

> 原文:[https://www.geeksforgeeks.org/c-destructors-question-3/](https://www.geeksforgeeks.org/c-destructors-question-3/)

像构造函数一样，一个类中可以有多个析构函数吗？
**(A)** 是
**(B)** 否

**回答:****(B)**

**说明:**一个类只能有一个析构函数。析构函数的签名始终是 *~ClassNam()* ，不能作为参数传递。
