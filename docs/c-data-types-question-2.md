以下哪一项不是 C 语言中的有效声明？

```cpp
1\. short int x;
```

```cpp
2\. signed short x;
```

```cpp
3\. short x;
```

```cpp
4\. unsigned short x;
```

**(A)** 3、4
**(B)**2
**(C)**1
**(D)**均有效

**回答:****(D)**

**说明:**均有效。前三个意思相同。第四个表示无符号。
