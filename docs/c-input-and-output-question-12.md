以下程序的输出？

```cpp
#include<stdio.h>
int main()
{
    printf("%d", printf("%d", 1234));
    return 0;
}
```

**(A)**12344
**(B)**12341
**(C)**11234
**(D)**41234

**答案:****(A)**

**解释:** printf()返回屏幕上成功打印的字符数。
