与 malloc 相比，以下哪一项是正确的？
1) new 是一个运算符，malloc 是一个函数
2) new 调用构造函数，malloc 不
3) new 返回适当的指针，malloc 返回 void *并且指针需要类型转换为适当的类型。

**(A)** 1 和 3
**(B)** 2 和 3
**(C)** 1 和 2
**(D)** 全部 1、2 和 3

**答案:****(D)**
