关于 C 语言中的“跳转语句”，以下哪一项是正确的？
**(A)** 转
**(B)** 继续
**(C)** 破
**(D)** 返回
**(E)** 以上全部。

**回答:****(E)**

**解释:**按照 C 标准，“一个跳转语句导致无条件跳转到另一个地方”。所以如果我们仔细看，所有的“goto”、“continue”、“break”和“return”都会使程序控制无条件地从一个地方跳到另一个地方。这就是为什么正确答案是 e .这道题的
