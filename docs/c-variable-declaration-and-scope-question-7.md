# C |变量声明和范围|问题 7

> 原文:[https://www . geesforgeks . org/c-variable-declaration-and-scope-question-7/](https://www.geeksforgeeks.org/c-variable-declaration-and-scope-question-7/)

```
int main()
{
  int x = 032;
  printf("%d", x);
  return 0;
}
```

**(A)**32
**(B)**0
**(C)**26
**(D)**50

**答案:****(C)**

**说明:**当一个常数值以 0 开头时，就认为是八进制数。因此 x 的值是 3*8 + 2 = 26
