假设下面的 C 变量声明

```cpp
int *A [10], B [10][10];
```

在下列表达式中

```cpp
I.   A [2]
II.  A [2][3]
III. B [1]
IV.  B [2][3] 
```

如果在 C 程序中用作赋值语句的左侧，哪个不会产生编译时错误？

**(A)** I、II、IV 仅
**(B)** II、III、IV 仅
**(C)** II、IV 仅
**(D)** IV 仅

**答案:****(A)**
