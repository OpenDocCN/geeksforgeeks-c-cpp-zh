**语法:**

```
??=define
Becomes,
#define

```

下面是三元图序列的基本实现示例:

## C++

```
// C++ implementation to 
// illustrate the example 
// of trigraph sequences

# include <iostream>
using namespace std;

??=define MSG "GeeksforGeeks"
??=define Program "C++"

// Driver Code
int main()
??< // Here ??< denotes {
    cout << "My message to " << MSG << endl;
    cout << "My program is " << Program;

    return 0;
??> // Here ??> denotes }
```

**Output:**

```
My message to GeeksforGeeks
My program is C++

```

下面是三元图序列的另一个例子:
