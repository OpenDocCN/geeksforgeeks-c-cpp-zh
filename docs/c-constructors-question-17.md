```cpp
#include<iostream>
using namespace std;

class Test
{
public:
   Test(Test &t) { }
   Test()        { }
};

Test fun()
{
    cout << "fun() Called\n";
    Test t;
    return t;
}

int main()
{
    Test t1;
    Test t2 = fun();
    return 0;
}
```

**(A)** fun()调用
**(B)** 空输出
**(C)** 编译器错误:因为复制构造函数参数是非常量

**答案:****(C)**
